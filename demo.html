Organizations Management

Fields: orgId, orgName, mnemonic, queriesKey, description, authorizationScheme, supportedBy, managedBy, serviceOffering, isActive

Key Features:
  • Side-by-side JSON editor
  • Authorization scheme logic (functional/ad_group)
  • Previous/Next navigation

Servers Management

Fields: hostName, domain, appLob, wfguid, appid, app_supported_by, app_managed_by, tso_managed_by, tso_supported_by, device_managed_by, device_supported_by, isActive, orgId

Key Features:
  • Organization mapping
  • Advanced filtering
  • Export functionality

Commands Management

Fields: orgId, commandId, commandLabel, description, isActive

Key Features:
  • Organization mapping
  • Search and pagination
  • Navigation between records

Dashboard Features

  • Overview statistics
  • Search & filtering
  • Pagination
  • No popup forms (all on new pages)
  • Export capabilities
  • Responsive design

To Run This Application:

  1. Install Node.js from nodejs.org
  2. Open terminal in project directory
  3. Run: npm install
  4. Run: npm start
  5. Open http://localhost:3000
